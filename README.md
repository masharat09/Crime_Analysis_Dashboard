🚔 Crime Analysis Dashboard

📌 Project Overview

    The Crime Analysis Dashboard is an interactive Power BI report designed to analyze crime incidents across different years, months, countries, weekdays, and time ranges. The dashboard helps identify crime trends, dangerous time periods, resolution rates, and geographical crime distribution to support data-driven decision-making.

📷 Dashboard Preview
https://drive.google.com/file/d/17P9VRQoxfUnof4IYKWLt2pdbmPWpaVg6/view?usp=drive_link

🎯 Objectives

    Analyze overall crime incidents.
    Compare crime trends across multiple years.
    Identify high-crime months and weekdays.
    Detect the most dangerous time ranges.
    Visualize crime distribution by country.
    Monitor crime resolution and unresolved cases.
    
🛠️ Tools & Technologies

    Power BI – Dashboard Development & Visualization
    Excel/CSV – Data Source
    DAX – Calculated Measures & KPIs
    
📊 Dashboard Features
  
1️⃣ KPI Cards
 
    Total Crimes
    Crime Resolution Rate
    Unresolved Crime Percentage
    Dangerous Crime Time Indicator
    
2️⃣ Crime Trend Analysis

    Crime by Year (2021–2023)
    Crime by Month
    Monthly Growth/Decline Indicators
    
3️⃣ Time-Based Analysis

    Crime by Time Range
    Identification of peak crime hours
    Dangerous time period insights
    
4️⃣ Weekly Analysis

    Heatmap showing crime occurrence by:
    Month
    Weekday
    
5️⃣ Geographic Analysis

    Crime distribution by country
    Interactive map visualization
    
6️⃣ Interactive Filters

    Dynamic slicers for easy exploration
    Cross-filtering across all visuals
